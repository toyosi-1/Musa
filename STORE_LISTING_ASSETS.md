1. **Use Real Data** (but anonymized):
   - Use placeholder names like "John Doe", "Jane Smith"
   - Use fake phone numbers: +234 XXX XXX XXXX
   - Use sample addresses

2. **Show Key Features**:
   - Highlight the main value proposition in each screenshot
   - Use annotations or captions if allowed

3. **Consistent Branding**:
   - Use the same theme (light or dark) across all screenshots
   - Maintain consistent status bar appearance

4. **Device Frames** (optional but recommended):
   - Use tools like Figma, Sketch, or online mockup generators
   - Add device frames for professional appearance

5. **Capture Tools**:
   - iOS: Xcode Simulator (Cmd+S to save screenshot)
   - Android: Android Studio Emulator (Screenshot button)
